3) financial-calculations.ts – Entnahmeberechnung
A) In calculateMaximumWithdrawal
function calculateMaximumWithdrawal(
  startingCapital: number,
  months: number,
  inflationAdjusted: boolean,
  portfolio: Strategy
): number {
  const { net } = getAnnualParamsForStrategy(portfolio);
  const monthlyReturn = Math.pow(1 + net, 1/12) - 1;

  // immer konstante nominale Entnahme berechnen
  return startingCapital * monthlyReturn / (1 - Math.pow(1 + monthlyReturn, -months));
}


👉 keine Abhängigkeit mehr von inflationAdjusted in der Formel (immer konstant nominal).
